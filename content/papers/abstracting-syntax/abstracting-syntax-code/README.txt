These are the Coq sources for the techincal report "Abstracting Syntax" by
Brian Aydemir, Stephanie Weirich, and Steve Zdancewic.

- Run 'make' to compile the Coq code here.
  The code is known to compile with Coq 8.1pl4.

- Open index.html in a web browser to get an overview of the files here.
  The documentation can be regenerated by running 'make documentation'.

- coqwc.mll is the source code to our modified version of the coqwc tool
  that comes with Coq.  The numbers we get out of our version make a bit
  more sense to us than what Coq's coqwc tool generates.  Run 'make coqwc'
  to build our version.
